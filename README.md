**ISBN Look-up Service **

A robust, Java-based utility designed to accurately scan, validate, and retrieve structured book data using International Standard Book Numbers (ISBN).

Key Features:

- Strict Data Validation: Parses and validates complex string/numeric inputs to ensure accurate processing.

- SOLID Principle Compliance: Engineered using SOLID object-oriented design principles to ensure the codebase is modular and easily extensible.

- Loosely Coupled Design: Utilizes interface-driven development. The core validation logic is completely decoupled from the data-fetching implementation, allowing APIs or databases to be swapped out seamlessly.

- Test-Driven: Comprehensive unit testing implemented via JUnit to guarantee zero-defect data processing.

- Fully Documented: Thoroughly commented using Javadocs to ensure clear internal documentation and rapid onboarding for other developers.
